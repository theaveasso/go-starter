### Installation
1. Clone the repository:
```
git clone https://github.com/theaveasso/go-starter
```
2. Navigation to the project directory:
```
cd go-starter
```
3. Install dependencies
```
go mod tidy
```

How to run the project
- Using `make`
```
// build
make build

// run
make run
```

- Using `air`
```
air
```
